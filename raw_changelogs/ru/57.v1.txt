UTCrypt v0.6FR4(57) от 2021-06-04

Main Window
- Добавлена заглушка на случай отсутствия уведомлений.
- Исправлено отображение иконки на переключателе отображения уведомлений в различных ситуациях.
Password Generator
- Улучшена разметка интрерфейса.
- Реализованы все параметры для генерируемого пароля.
- Улучшено информирование о неправильных вводимых значениях.
- Добавлена возможность генерации пароля фиксированной длины.
- Оптимизация контента.
- Оптимизация генерации с огромным увеличением её скорости.
- Добавлена функция копирования в буфер обмена.

Styles
- CheckBox и Switch переделаны под нормальную ориентацию - больше не разворачивают знаки препинания в контенте, их код облегчён.
- Все цвета в CheckBox теперь ссылаются на динамические ресурсы (не будет на голубой теме фиолетовой галочки).
- Switch получил улучшене разметки.

Etc
- Возможно исправлена неизвестная ошибка при проверке наличия обновлений на самой актуальной версии.
- Добавлен тип шифрования "Без шифрования". Он лишь пропускает сообщения через выбранные фильтры обработки, без шифрования.
- Добавлена отладочная команда для открытия консоли. Нужна для тестов функционирования гибридного приложения из WPF и CMD.
